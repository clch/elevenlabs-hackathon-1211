# Interactive Alarm Clock (Next.js + TypeScript)

This is a minimal Next.js app for an **interactive alarm clock** that:

- Starts with ElevenLabs agent `agent_1301kc88drzve9naq1aew5nfbvdv`
- After **30 seconds**, switches to `agent_1901kc8anst7evjrf7j61fs0sc99`

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
